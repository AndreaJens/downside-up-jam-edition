# A game made in 90 minutes

For the 90-minutes-jam at the Godot Camp 2026 in Karlsruhe, I was challenged to make a full game in 90 minutes.

The theme was "Switching Perspectives". This is the source code of **Downside Up**, the very simple game I made in that time... using all the dirtiest tricks in my toolkit.

The controls are simple (move with WASD or arrows, switch plane with UP or DOWN when on the ground, survive until you can).

It was a challenge to finish this in 90 minutes, but -hey- at least it works!
